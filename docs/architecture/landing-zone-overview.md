#  Landing  Zone  Overview

This  landing  zone  implements  a  **hub-and-spoke,  policy-driven,  identity-centric**  Azure  platform.

##  Core  Pillars

-  **Management  Groups**:  Organize  subscriptions  and  apply  policies.
-  **Networking**:  Hub-and-spoke  with  shared  services  in  the  hub.
-  **Identity**:  AAD  groups,  RBAC,  and  managed  identities.
-  **Governance**:  Azure  Policy,  initiatives,  and  tagging.
-  **Observability**:  Log  Analytics,  diagnostics,  and  Sentinel.
-  **Automation**:  Subscription  vending  and  operational  runbooks.
